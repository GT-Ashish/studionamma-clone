# 🎬 StudioNamma Clone

A responsive front-end clone of the StudioNamma website built using HTML, CSS, and JavaScript. This project recreates the look and feel of the original platform with a modern UI, smooth layouts, and interactive design elements.

## 🚀 Live Demo

🔗 Live Website: https://gt-ashish.github.io/studionamma-clone/

## 📌 Features

- Responsive design for different screen sizes
- Modern landing page layout
- Hero section with engaging visuals
- Navigation bar with clean styling
- Interactive UI elements
- Smooth hover effects and transitions
- Organized and reusable code structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
